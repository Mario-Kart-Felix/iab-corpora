			<p>Mindy I would do this:</p>
<p>1. Open the file &#8220;12-vintage-cars-vector-art-keepdesigning.eps&#8221; with Illustrator. </p>
<p>2. Click to select the car vector you want and copy it to the clipboard (Edit>Copy)</p>
<p>3. Create a new file in Illustrator (File>New&#8230;), making it the final print size you want.</p>
<p>4. Paste the car vector into the new file (Edit>Paste)</p>
<p>5. Use Illustrator&#8217;s scale tool to enlarge the vector as big as you want.</p>
<p>6. Print away! (File>Print)</p>
<p>Hope that helps. Here&#8217;s a good video on using Illustrator&#8217;s scale tool: <a href="http://www.youtube.com/watch?v=ka2UbQQ7APM&#038;feature=related" rel="nofollow">http://www.youtube.com/watch?v=ka2UbQQ7APM&#038;feature=related</a></p>
