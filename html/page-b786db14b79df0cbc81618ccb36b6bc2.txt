						<div itemprop="description">
							<p>  	Following the unveiling of its <a href="http://www.themotorreport.com.au/51477/2012-kia-rio-hatch-geneva-motor-show">five-door hatch counterpart in Geneva last month</a>, Kia has pulled the covers back from the <a href="http://www.themotorreport.com.au/tag/kia-rio">2012 Kia Rio Sedan</a>.</p>  <p>  	In the US market, the new Rio Sedan will be offered in LX, EX and range-topping SX trim grades, but Australian variant names will likely mirror the local S, Si and SLi badges.</p>  <p>  	Power for the US models will come from a direct-injected 100kW 1.6 litre four-cylinder petrol engine, although engines ranging from a 51kW three-cylinder 1.1 litre diesel to a 80kW 1.4 litre four-cylinder petrol engine have been slated for international models.</p>  <div class="inarticle-ad" style="text-align: center;">		<div id="ad-521d504fa23d7" style="height: 250px;">
			<noscript><a href="http://ad.au.doubleclick.net/jump/tmr.news/details;car=kiario;sz=300x250;tile=3;cat=sedan;ord=88856532?" target="_blank"><img src="http://ad.au.doubleclick.net/ad/tmr.news/details;car=kiario;sz=300x250;tile=3;cat=sedan;ord=88856532?" border="0" alt=""/></a></noscript>
		</div>
</div><p>  	Paired with either a six-speed manual or six-speed automatic transmission, Kia says the 1.6 litre petrol engine will return city-driving figures of 7.8 l/100km, and 5.8 l/100km on the highway.</p>  <p>  	
	<div class="inline-image-right" style="float: right; width: 200px">
		<a href="/content/image/2/0/2012_kia_rio_sedan_04-4db109fabaddc.jpg" class="lightbox" title=""><img src="/content/image/2/0/2012_kia_rio_sedan_04-4db109fabaddc-200x200.jpg" alt="2012 kia rio sedan 04" class="small"/></a>
			</div>
</p>  <p>  	In a first for Kia, the Rio is also equipped with ISG (idle stop and go) technology, switching off the engine when the vehicle is not in motion.</p>  <p>  	The 2012 Rio Sedan is larger than the model it will soon replace. Its wheelbase is 70mm longer, and the body measures 4366mm long overall.</p>  <p>  	It is also 25mm wider, offering increased interior space with 45mm more front legroom.</p>  <p>  	Standard safety features across the Rio range will include three-point seatbelts all-round, electronic stability control, four-wheel ABS, hill-start assist and vehicle stability management. </p>  <h3>  	Australia</h3>  <p>  	Although the sedan is expected to join the hatch when it launches in Australia later this year, local specifications will be revealed closer to launch.</p>  <p>

		</div>

	$(".emoticons-container a").click(function(evnt) {
		evnt.preventDefault();

		var text = $(this).attr("data-emoticon");
		var textarea = document.getElementById("comment_text");

		var currSelectionStart = textarea.selectionStart;
		var currText = textarea.value;
		currText = currText.substring(0, textarea.selectionStart) +
		           text + currText.substring(textarea.selectionEnd);
		textarea.value = currText;

		$(textarea).focus();
		textarea.selectionStart = currSelectionStart+text.length;
	});

	$('#comment-form').ajaxForm({
		type: 'post',
		beforeSubmit: function(request) {
			$("#comment-error").empty().hide();

			$("#comment-form-wrapper .indicator").hide().html(loading_html).fadeIn();
			$("#comment-form input[type=image]").prop("disabled", true);
		},
		success: function(data) {
			$("#comment-reply-to").empty();
			$("#comment-form input[type=image]").prop("disabled", false);

			if (data.indexOf("ERROR:") === 0) {
				data = data.substr(6);
				$("#comment-error").html(data).show();
			} else {
				var parentId = $("#comment_parent_id").val();
				if (parentId != "" ) {
					var container = $("#comment_" + parentId).parent("li");

					// figure out what level the new comment is at
					var level = container.parents("ul").get(0).className;
					level = level.split("-")[2];
					level = (1*level) + 1;

					container.append("<ul class=\"comments-level-"+level+"\"><li>"+data+"</li></ul>");
				} else {
					$("#comments").append("<ul><li>" + data + "</li></ul>");
				}

				// if they've unchecked "Remember me", make sure to reset that part of the
				// form as well
				if (!$("#register_remember").prop("checked")) {
					$("#register_remember").removeAttr("checked");
					$("#register_name").attr("value", "");
					$("#register_email").attr("value", "");
				}

				FB.XFBML.parse();
				$("#comment-form").resetForm();

				var url = window.location.href.replace(window.location.hash, "");
				var regex = /id="comment_([0-9]+)"/i;
				var m = regex.exec(data);
				if (m.length > 1) {
					url += "#comment_"+m[1];
					window.location.href = url;
				}
			}
		}
	});
