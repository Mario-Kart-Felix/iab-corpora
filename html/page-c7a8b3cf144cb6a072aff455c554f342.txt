<H3><FONT FACE=VERDANA COLOR="008040">Questions and Answers</FONT></H3>

<BLOCKQUOTE>

<FONT FACE=VERDANA SIZE=2>

<TABLE BORDER=0 WIDTH=400>
<TR>
<TD ALIGN=LEFT VALIGN=TOP>
<FONT FACE=VERDANA SIZE=2>

<P><B><U>Cadillac Sedan Deville Has No Heat</U></B></P>

</FONT>
<P><FONT FACE=VERDANA SIZE=3 COLOR="FF0000"><B>Q. </B></FONT>
<FONT FACE=VERDANA SIZE=2>
I have a 1994 Cadillac Sedan Deville, I changed the antifreeze, checked all of the hoses, put in new thermostat and my heater still does not put out warm air, but it is moving freely through the engine what do I need to do now?</P>
 
<P>Thanks,<BR>
Elisabeth</P>

<TD ALIGN=LEFT VALIGN=TOP><IMG SRC="http://0.tqn.com/d/autorepair/1/0/Z/L/mechanic.jpg" ALT="Cadillac Sedan Deville Has No Heat"></A></TD></TR>
</TABLE>

<P><B><FONT FACE=VERDANA SIZE=3 COLOR="FF0000">A. </FONT></B>I'm assuming air flow through the Electronic Climate Control (ECC) is normal, the air is just not getting hot.</P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'>If that is, indeed, the case, then I believe the problem may be with the Air Mixture Valve. This valve, door actually, directs incoming air either through the heater core to be warmed or around it.</P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'>This is controlled by the Air Mixture Valve Control Motor in the Climate Control Center (CCC). You can use the ECC self diagnostics to access the DTC to confirm this, or any other problem.</P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'>To enter diagnostic mode, proceed as follows:</P>

<OL>
<LI>Turn ignition "ON."
<LI>Depress the "OFF" and "WARMER" buttons on the Climate Control Center <B>(CCC)</B> simultaneously and hold until the segment check appears on the Instrument Panel Cluster <B>(IPC)</B>.
</OL></P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'><B>TROUBLE CODE DISPLAY</B>:<BR>
After entering On-Board Diagnostics, a segment check is automatically performed and then any trouble code stored in computer memory will be displayed. Codes may be stored for the Powertrain Control Module <B>(PCM)</B>, Instrument Panel Cluster <B>(IPC)</B>, Air Conditioning Programmer <B>(ACP)</B>, Supplemental Inflatable Restraint <B>(SIR)</B>, Traction Control System <B>(TCS)</B> or Road Sensing Suspension <B>(RSS)</B> systems.</P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'>Each trouble code consists of the system abbreviation: (E: Engine, PCM), (I: IPC), (A: ACP), (R: SIR), (T: TCS), (S: RSS), a three digit code identifier, and the letter C or H. All codes for each system are displayed together, in the order of their three digit number (lowest first). The final digit of the code, a "C" or "H" is used to indicate whether the code is Current or History. A "C" means the fault was still present the last time the diagnostic test was run, while "H" means the failure was not present the last time the code was tested for.</P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'>If no codes are present for a system, a "NO X CODE" message (with "X" being the system i.e. E, I, A, R, T or S) will be displayed. If the communication line to a component is not operating, a "NO X DATA" message will be displayed, indicating that the IPC could not communicate with that system.</P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'><B>SELECTING THE SYSTEM</B>:<BR>
After trouble codes have been displayed, a specific vehicle system PCM, IPC, ACP, SIR, TCS, RSS may be selected for testing.</P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'>The first available system will be automatically displayed (i.e. PCM?). While selecting the system to test, any of the following actions may be taken:

<OL>
<LI>Depressing the "HI" button on the CCC will select the displayed system for testing.
<LI>Depressing the "LO" button on the CCC will display the next available system selection. This allows the display to be stepped through all system choices. This list of systems can be repeated following the end of the system list.
<LI>Depressing the "OFF" button the CCC will stop the system selection process and return the display to the beginning of the trouble code sequence.
</OL>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'><B>SELECTING THE TEST TYPE</B>:<BR>
Having selected a system, the first available test type will be displayed (i.e. PCM DATA?, PCM CLEAR CODES?). While selecting a specific test type, any of the following actions may be taken:</P>

<OL>
<LI>Depressing the "HI" button on the CCC will select the displayed test type. At this point the first of several specific tests will appear.
<LI>Depressing the "LO" button on the CCC will display the next available test type for the selected system. This allows the display to be stepped through all available test type choices. This list of test types can be repeated following the display of the last test type.
<LI>Depressing the "OFF" button on the CCC will stop the test type selection process and return the display to the next available system selection.
</OL>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'><B>CLEAR CODES</B> <BR>
Selection of "CLEAR CODES?" test will result in the message "CODES CLEARED" being displayed along with the selected systems name. This message will appear for 3 seconds and then return to the next available test type.</P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'><B>EXITING DIAGNOSTIC MODE</B>:<BR>
To get out of the diagnostic mode anytime during the test, depress the "AUTO" or "DEFOG" button, or turn the ignition switch "OFF." Trouble codes are not erased if this is done prior to selecting "CODES CLEARED."</P>

<P CLASS=MSOBODYTEXT STYLE='TEXT-INDENT:. 2IN'><B>ECC Trouble Code Descriptions:</B></P>

<UL>
<B><LI>DTC A010</B> - Outside Temperature Sensor Circuit
<B><LI>DTC A011</B> - High Side Temperature Sensor Circuit
<B><LI>DTC A012</B> - Low Side Temperature Sensor Circuit
<B><LI>DTC A013</B> - In-Car Temperature Sensor Circuit
<B><LI>DTC A015</B> - Sunload Sensor Circuit
<B><LI>DTC A037</B> - Loss of IPC Data
<B><LI>DTC A040</B> - Air Mix Valve (Door) Circuit Problem
<B><LI>DTC A047</B> - Low Refrigerant Charge
<B><LI>DTC A048</B> - Low Refrigerant Pressure
<B><LI>DTC A050</B> - HVAC - Coolant Temperature Too High
</UL>
