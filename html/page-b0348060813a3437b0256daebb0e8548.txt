						<div itemprop="description">
							<p>  	<strong>HOLDEN CRUZE HATCHBACK</strong></p>  <p>  	Its coming has been known for as long as its sedan sibling has existed in Australia, but at last the covers have come off: this is the <a href="http://themotorreport.com.au/tag/holden-cruze">2012 Holden Cruze Hatchback</a> - albeit, in showcar form and wearing its international Chevrolet front-end.</p>  <p>  	Designed and built in Australia, the Cruze Hatchback will be marketed as a global model for GM - wearing the Chevrolet badge and face in most markets - with assembly for the Australian market to be carried out right here at home from late 2011.</p>  <div class="inarticle-ad" style="text-align: center;">		<div id="ad-521d35a704b49" style="height: 250px;">
			<noscript><a href="http://ad.au.doubleclick.net/jump/tmr.news/details;car=holdencruze;sz=300x250;tile=3;ord=85303658?" target="_blank"><img src="http://ad.au.doubleclick.net/ad/tmr.news/details;car=holdencruze;sz=300x250;tile=3;ord=85303658?" border="0" alt=""/></a></noscript>
		</div>
</div><p>  	The Cruze sedan is expected to switch to local production in the first quarter of next year.</p>  <p>  	
	<div class="inline-image-center">
	<a href="/content/image/2/0/2012_holden_cruze_hatchback_01-4c811f39e7995.jpg" class="lightbox"><img alt="The 2012 Holden Cruze Hatchback was designed and built in Australia." src="/content/image/2/0/2012_holden_cruze_hatchback_01-4c811f39e7995-460x272.jpg" class="small"/></a>
			<div class="caption">The 2012 Holden Cruze Hatchback was designed and built in Australia.</div>
		</div>
</p>  <p>  	While the Cruze Hatchback pictured here is a showcar only, Holden&#39;s design boss Richard Ferlazzo said today that it is a precursor to a production model, designed over 12 months alongside GM designers from around the world.</p>  <div class='blockquote clearfix'><div class='blockquote-left'> </div><div class='blockquote-center'><blockquote>  	<p>  		&quot;Our brief was to create an alternative body style to the traditional sedan that retained the same dynamic proportions that has made Cruze such a success around the world,&rdquo; Mr Ferlazzo said.</p>  	<p>  		&ldquo;We are very proud of the result which will be tailored further for the Australian market and given an appearance consistent with the rest of the Holden portfolio next year.&rdquo;</p>  </blockquote></div></div><div class='clearfix blockquote-right'> </div>  <p>  	
	<div class="inline-image-center">
	<a href="/content/image/2/0/2012_holden_cruze_hatchback_02-4c811f3d419a9.jpg" class="lightbox"><img alt="" src="/content/image/2/0/2012_holden_cruze_hatchback_02-4c811f3d419a9-460x272.jpg" class="small"/></a>
		</div>
</p>  <p>  	Despite its seemingly smaller shape, Holden says luggage capacity in the rear of the Cruze Hatchback is &quot;close&quot; to the 400 litre capacity of the Cruze sedan, and the rear seats feature a 60/40 split fold.</p>  <p>  	Holden and Chevrolet have released little else at this stage, although the full details of this Cruze Hatchback showcar will likely be revealed at this month&#39;s Paris Auto Show.</p>

	$(".emoticons-container a").click(function(evnt) {
		evnt.preventDefault();

		var text = $(this).attr("data-emoticon");
		var textarea = document.getElementById("comment_text");

		var currSelectionStart = textarea.selectionStart;
		var currText = textarea.value;
		currText = currText.substring(0, textarea.selectionStart) +
		           text + currText.substring(textarea.selectionEnd);
		textarea.value = currText;

		$(textarea).focus();
		textarea.selectionStart = currSelectionStart+text.length;
	});

	$('#comment-form').ajaxForm({
		type: 'post',
		beforeSubmit: function(request) {
			$("#comment-error").empty().hide();

			$("#comment-form-wrapper .indicator").hide().html(loading_html).fadeIn();
			$("#comment-form input[type=image]").prop("disabled", true);
		},
		success: function(data) {
			$("#comment-reply-to").empty();
			$("#comment-form input[type=image]").prop("disabled", false);

			if (data.indexOf("ERROR:") === 0) {
				data = data.substr(6);
				$("#comment-error").html(data).show();
			} else {
				var parentId = $("#comment_parent_id").val();
				if (parentId != "" ) {
					var container = $("#comment_" + parentId).parent("li");

					// figure out what level the new comment is at
					var level = container.parents("ul").get(0).className;
					level = level.split("-")[2];
					level = (1*level) + 1;

					container.append("<ul class=\"comments-level-"+level+"\"><li>"+data+"</li></ul>");
				} else {
					$("#comments").append("<ul><li>" + data + "</li></ul>");
				}

				// if they've unchecked "Remember me", make sure to reset that part of the
				// form as well
				if (!$("#register_remember").prop("checked")) {
					$("#register_remember").removeAttr("checked");
					$("#register_name").attr("value", "");
					$("#register_email").attr("value", "");
				}

				FB.XFBML.parse();
				$("#comment-form").resetForm();

				var url = window.location.href.replace(window.location.hash, "");
				var regex = /id="comment_([0-9]+)"/i;
				var m = regex.exec(data);
				if (m.length > 1) {
					url += "#comment_"+m[1];
					window.location.href = url;
				}
			}
		}
	});
