       				<h1>Here's the buzz: Auto insurance for electric cars</h1>          			
        				        			<p class="byline" style="padding-bottom:9px;">
	        				
															
	        				By Jeffrey Steele, Insure.com - Last updated: Dec. 28, 2010	        			</p>
						
							<div class="dotted_line"></div> 
			<p>Some are calling 2011 the "Year of the Electric Vehicle." The most ballyhooed new electric cars, the Chevy Volt and Nissan Leaf, are generating a current of excitement, and other electrics will be unveiled soon.</p>
<p>Electric cars are powered by a battery that is recharged by plugging it into a charging station. They promise to reduce pollution and our dependence on oil and gas.</p><p class="help_text"><a onclick="dcsMultiTrack('DCS.dcsuri','/linked-from-article-car.html');" href="http://www.insure.com/autonewmodel/geo?r=160262509&var1=textAB%3Dv3_textlink" title="Insure.com" rel="nofollow">[Let Insure.com help you find affordable car insurance now.]</a></p>
<p><img style="margin: 5px; float: right;" title="electric car" src="/imagesvr_ce/5438/electric-car.jpg" border="0" alt="electric car" width="300" height="225" />But will <a href="http://www.insure.com/">car insurance rates</a> for electric cars be more expensive than those for conventional automobiles? According to James Bell, executive market analyst for Kelley Blue Book and KBB.com in Irvine, Cal., there are very good reasons for premiums on electric vehicles to be <em>lower</em> priced.</p>
<h3>How electric cars lead to low car insurance rates</h3>
<p>That assessment is based initially on similarities with the first hybrid cars. When hybrids were unveiled, they were rated as less risky than other vehicles, Bell says. The early hybrids were not sporty and tended to attract buyers who were intelligent, earth-minded and conservative. In short, they were the opposite of the reckless, risk-taking motorist who gets higher <a href="http://www.insure.com/car-insurance/">auto insurance quotes</a>.</p>
<p>Except for the very limited edition, sporty Tesla, this coming generation of electric vehicles is much the same. Electrics are likely to attract the same brand of stable, conservative, safe-driving buyers to which early hybrids appealed.</p>
<p>Second, this wave of electric vehicles will likely be used for commuting and around-town errand running, Bell says. Range limitations and a lack of public areas for charging batteries mean they won't be taken on long trips.</p>
<p>"So when you step back and consider how the electric vehicles are going to be used, they're going to be used conservatively," Bell says.</p>
<p>Third, although electric vehicles are associated with very new technology, they are based on a mechanics that are at least a century old and very simple.</p>
<p>"Electric vehicles have very few moving parts," Bell says. "As far as reliability goes, that's not a problem. This is an existing technology seen in golf carts. The mechanical part of an electric car is very tried and tested."</p>
<p>The simple mechanics and comparative dearth of moving parts in electrics compared with standard engine vehicles may lead to less complex and costly repairs, resulting in less claims expense to insurance companies, Bell says.</p>
<p>Finally, electric cars will face less overall exposure to accidents because they will be driven considerably fewer miles. "The combination of factors -- a safe driver, around-town driving and less complexity than ordinary cars or hybrids -- would seem to suggest they would have lower car insurance rates than other cars," Bell says.</p>
<h3>Some juries are still out</h3>
<p>The view from one major car insurance company is cautious. At Bloomington, Ill.-based State Farm Insurance, spokesperson Dick Luedke says it's still too early to determine <a href="http://www.insure.com/car-insurance/">auto insurance</a> rates for electric cars.</p>
<p>"We have a make-and-model rating system based on our claims data," he says. "We can see the makes and models that generate the most cost to us as an insurer for each type of claim. For instance, some vehicles do better than others at protecting the occupants of the car, and that shows up in our data. Some right front fenders are more expensive to replace than others, and <em>that </em>shows up in our data."</p>
<p>The issue with electric cars is that there is not yet sufficient data to make any conclusions about whether they generate insurance claim costs that are more or less than average, he says. And that's not unusual. Any time a new make and model is introduced, the company has to collect enough data to make conclusions. It's a process that takes time, Luedke says.</p>
<p>Nonetheless, Bell anticipates that "the acceptance of electric cars should be based on the fact they exist and are an option for consumers in dealerships today, rather than on how high or low their car insurance rates are."</p></p>

var AutoNewModel = {
	getErrorMessage: function(zipcode_type)
	{
		var ca_error_msg = 'Please enter a valid Canada postal code.';
		var us_error_msg = 'Please enter a valid US zip code.';
		var error_msg = 'Please enter a valid US or Canada postal code.';
		if(zipcode_type == 'US')
		{
			return us_error_msg;
		}
		else if(zipcode_type == 'Canada')
		{
			return ca_error_msg;
		}
		else
		{
			return error_msg;
		}
	},
	
	showFormErrors: function(error, form)
	{
		var newmodel = this;
		newmodel.clearAllFormErrors(form);
		if(error[0].field == 'form')
		{
			newmodel.addErrorFlag(form, error[0].message);
		}
		else
		{
			for(x in error)
			{
				var location = '';
				if(error[x].field == 'zip' || error[x].field == 'insured')
				{
					location = form + ' input[name="' + error[x].field + '"]';
					$(location).addClass('formError');
					newmodel.addErrorFlag(location, error[x].message);
					return false;
				}
				else
				{
					newmodel.clearAllFormErrors(form);
					location = form;
					newmodel.clearInputError(form);
				}
			}
		}
	},
	
	clearAllFormErrors: function(form)
	{
		jQuery(form).find('.errorFlag').remove('.errorFlag');
	},
	
	addErrorFlag: function(location, message)
	{
		var newmodel = this;
		var field = $(location).parent('fieldset');
		var loc_field_type = $(location).get(0).tagName;
		var input_height = 0;
		var input_parent_height = 0;
		var error_height = 0;
		if(loc_field_type == 'INPUT')
		{
			var input_parent_height = parseInt($(location).prev(1).outerHeight());
			var input_pos = $(location).position();
			var input_top = parseInt(input_pos.top);
			var input_height = parseInt($(location).outerHeight());
			if(!isNaN(input_height) && !isNaN(input_parent_height))
			{
				if(input_height > input_parent_height)
				{
					var error_height = input_height - input_parent_height;
				}
				else
				{
					var error_height = input_parent_height + input_height;	
				}
			}
		}
		jQuery(field).prepend('<div class="errorFlag"><div class="paddingWrapper"><a href="void()" class="close">x</a><p>' + message + '</p></div></div>');
		jQuery(location).siblings('.errorFlag').children('.paddingWrapper').children('.close').click(function (j) {
			j.preventDefault();
			newmodel.clearInputError(location);
		});
	},
	
	clearInputError: function(location)
	{
		jQuery(location).siblings('.errorFlag').remove();
		jQuery(location).removeClass('formError');
	},
	
	isValidZipCode: function(zip)
	{
		var ajxResponse = '';
		var isValid = /^[0-9]{5}(?:-[0-9]{4})?$/.test(zip);
		var host = 'http://' + document.domain;
		if (isValid)
		{
			jQuery.ajax({
				async: false,
				cache: false,
				url: host + '/autonewmodel/verifyzip',
				type: 'GET',
				data: 'zip=' + zip,
				dataType: 'text',
				success: function (result) {
					ajxResponse = result;
				},
				error: function () {
					return false;
				}
			});
			if(ajxResponse == 'invalid')
			{
				return false;
			}		
			return true;
		}
		else
		{
			return false;
		}
	},
	
	isValidCanadaZipCode: function(zip)
	{
		if(zip != '')
		{
			zip = zip.toUpperCase();
		}	
		var isValid = /^[ABCEGHJKLMNPRSTVXY][0-9][A-Z][\s]?[0-9][A-Z][0-9]$/.test(zip);
		if (isValid)
		{
			return true;
		}
		else
		{
			return false;
		}
	},
	
	sendAlert: function(xhr_request, xhr_status, post_data, quote_loc)
	{
		var newmodel = this;
		var debug_data = '';
		if( typeof (post_data) == 'string')
		{
			debug_data = post_data;
		}
		else
		{
			if(post_data.state != 'undefined' && post_data.state != null)
			{
				debug_data += '&state='+post_data.state;
			}
			if(post_data.zip != 'undefined' && post_data.zip != null)
			{
				debug_data += '&zip='+post_data.zip;
			}
			if(post_data.insured != 'undefined' && post_data.insured != null)
			{
				debug_data += '&insured='+post_data.insured;
			}
		}
		
		jQuery.ajax({
		type: 'GET',
		url: 'http://' + document.domain + '/autonewmodel/emailalert',
		data: 'errorDescription=' + newmodel.trimWhiteSpaces(xhr_request) +
		'&ajxStatus=' + xhr_status +
		'&where=' + quote_loc + 
		'&product=QTP auto' +
		debug_data 
		});				
	},
	
	trimWhiteSpaces: function(str)
	{
		if(typeof str != 'undefined')
		{
			return str.replace(/^\s+|\s+$/g, '');
		}
	}
