Their cars are typically not on the road every day and generally used for limited pleasure driving, car shows and club events. If you own a classic car, it&rsquo;s a good idea to shop around for affordable <a href="http://www.insure.com/">car insurance rates</a>. You may be able to receive a good deal from your current car insurance company or you could check out a specialty car insurer.</p>

<p>"Whether it's a 1932 Alfa Romeo or a 1962 Ferrari, antique and classic car owners should purchase auto insurance coverage for their collectible separate and apart from the policy that covers their regular-use vehicle," said Michael Barry, spokesperson for the Insurance Information Institute (III), in a statement. "While regular-use vehicles depreciate in value over the years, many classic cars appreciate in value and have to be insured accordingly."</p>
<h3>How car insurance companies define a classic car</h3>
<p>Exactly what qualifies as a &ldquo;classic car" varies among organizations and <a href="http://www.insure.com/">car insurance companies</a>. The Classic Car Club of America defines classic cars, American or foreign- built, as those produced between 1925 and 1948. They were generally built in limited quantities and expensive when new. III defines classic and antique cars as those manufactured 30 or more years ago.</p>
<p><a href="http://www.heacockclassic.com/">Heacock Classic Cars</a>, a division of Sebring, Fla.-based Heacock Insurance Group, Inc., an independent insurance company in business since 1922, expands the definition further. It defines "antique" cars as those manufactured more than 25 years ago, while "classic" cars are those built 15 to 25 years ago. Cars less than 25 years old generally fall into the insurer&rsquo;s "late model" or "exotic" designations, says Stacey Heacock, national sales manager for the insurer.</p>
<p>The market for classic cars isn&rsquo;t small. More than 6 million individuals are classic car collectors and 25 million are classic car enthusiasts, according to AutoTraderClassics.com, an online marketplace dedicated to the classic car sector.</p>
<h3>Classic car insurance policies</h3>
<p>Insuring a classic car is significantly different than getting a <a href="http://www.insure.com/">car insurance quote</a> for the family automobile. Many of these cars are rare and their owners shower them with devotion and expensive features.</p>
<p>"Collector car owners usually have an emotional attachment to their vehicles, which is not the case with the typical family vehicle," says Heacock, who oversees the collector car insurance market for more than 2,000 independent insurance agencies in the United States. "These are cars that owners have either lusted over since they were a teenager or spent hours of blood, sweat and tears restoring or repairing. As a result, the owner wants to be assured the insurance value at claim time will be equal to the full value of the car."</p>
<p>Classic car insurance policies also differ significantly from standard policies when it comes to how a loss is settled after an accident or theft, Heacock says.</p>
<h3>"Actual cash value" vs. "agreed value"</h3>
<p><a href="http://www.insure.com/">Car insurance companies</a> value standard cars based on "actual cash value," which takes depreciation into account. A collector car insurance policy, however, is usually written on an "agreed value" basis, meaning the owner will be paid the full value of a vehicle if it's lost or stolen, Heacock says.</p>
<p>The "agreed value" insurance agreement applies to physical damage caused by collision, fire, theft, vandalism or other types of damage, Heacock says.</p>

<p><a href="http://www.insure.com">Car insurance quotes</a> for classic cars vary somewhat depending on the value, age, type, and use of the vehicle, Heacock says. The typical savings for collector car insurance rates can be anywhere from one-third to one-half of the insurance rate for the family auto, she adds.</p>
<p>Though classic cars can cost less to insure than a standard vehicle, your classic car policy is likely to have restrictions. They can include:</p>
<ul>
<li>Limits on how many miles you drive the car each year</li>
<li>Mandates on how the car is stored (usually in a secured garage)</li>
<li>Restrictions on who may drive the vehicle</li>
</ul>
<p>For example, Heacock&rsquo;s standard classic car policies offer annual mileage options of 1,000, 3,000 or 6,000 miles.</p>
<p>"The insurance premium needs to reflect the limited use and better care," Heacock says.&nbsp;</p>

	getErrorMessage: function(zipcode_type)

		var error_msg = 'Please enter a valid US or Canada postal code.';

	showFormErrors: function(error, form)
	{
		var newmodel = this;
		newmodel.clearAllFormErrors(form);
		if(error[0].field == 'form')
		{
			newmodel.addErrorFlag(form, error[0].message);
		}
		else
		{
			for(x in error)
			{
				var location = '';
				if(error[x].field == 'zip' || error[x].field == 'insured')
				{
					location = form + ' input[name="' + error[x].field + '"]';
					$(location).addClass('formError');
					newmodel.addErrorFlag(location, error[x].message);
					return false;
				}
				else
				{
					newmodel.clearAllFormErrors(form);
					location = form;
					newmodel.clearInputError(form);
				}
			}
		}
	},
	
	clearAllFormErrors: function(form)
	{
		jQuery(form).find('.errorFlag').remove('.errorFlag');
	},
	
	addErrorFlag: function(location, message)
	{
		var newmodel = this;
		var field = $(location).parent('fieldset');
		var loc_field_type = $(location).get(0).tagName;
		var input_height = 0;
		var input_parent_height = 0;
		var error_height = 0;
		if(loc_field_type == 'INPUT')
		{
			var input_parent_height = parseInt($(location).prev(1).outerHeight());
			var input_pos = $(location).position();
			var input_top = parseInt(input_pos.top);
			var input_height = parseInt($(location).outerHeight());
			if(!isNaN(input_height) && !isNaN(input_parent_height))
			{
				if(input_height > input_parent_height)
				{
					var error_height = input_height - input_parent_height;
				}
				else
				{
					var error_height = input_parent_height + input_height;	
				}
			}
		}
		jQuery(field).prepend('<div class="errorFlag"><div class="paddingWrapper"><a href="void()" class="close">x</a><p>' + message + '</p></div></div>');

			j.preventDefault();
			newmodel.clearInputError(location);
		});
	},
	
	clearInputError: function(location)
	{
		jQuery(location).siblings('.errorFlag').remove();
		jQuery(location).removeClass('formError');
	},
	
	isValidZipCode: function(zip)
	{
		var ajxResponse = '';
		var isValid = /^[0-9]{5}(?:-[0-9]{4})?$/.test(zip);
		var host = 'http://' + document.domain;
		if (isValid)
		{
			jQuery.ajax({
				async: false,
				cache: false,
				url: host + '/autonewmodel/verifyzip',
				type: 'GET',
				data: 'zip=' + zip,
				dataType: 'text',
				success: function (result) {
					ajxResponse = result;
				},
				error: function () {
					return false;
				}
			});
			if(ajxResponse == 'invalid')
			{
				return false;
			}		
			return true;
		}
		else
		{
			return false;
		}
	},
	
	isValidCanadaZipCode: function(zip)
	{
		if(zip != '')
		{
			zip = zip.toUpperCase();
		}	
		var isValid = /^[ABCEGHJKLMNPRSTVXY][0-9][A-Z][\s]?[0-9][A-Z][0-9]$/.test(zip);
		if (isValid)
		{
			return true;
		}
		else
		{
			return false;
		}
	},
	
	sendAlert: function(xhr_request, xhr_status, post_data, quote_loc)
	{
		var newmodel = this;
		var debug_data = '';
		if( typeof (post_data) == 'string')
		{
			debug_data = post_data;
		}
		else
		{
			if(post_data.state != 'undefined' && post_data.state != null)
			{
				debug_data += '&state='+post_data.state;
			}
			if(post_data.zip != 'undefined' && post_data.zip != null)
			{
				debug_data += '&zip='+post_data.zip;
			}
			if(post_data.insured != 'undefined' && post_data.insured != null)
			{
				debug_data += '&insured='+post_data.insured;
			}
		}
		
		jQuery.ajax({
		type: 'GET',
		url: 'http://' + document.domain + '/autonewmodel/emailalert',
		data: 'errorDescription=' + newmodel.trimWhiteSpaces(xhr_request) +
		'&ajxStatus=' + xhr_status +
		'&where=' + quote_loc + 
		'&product=QTP auto' +
		debug_data 
		});				
	},
