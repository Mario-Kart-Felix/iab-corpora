						<div itemprop="description">
							<p>  	Nissan has revealed a new hybrid version of its <a href="http://www.themotorreport.com.au/tag/nissan-pathfinder">Pathfinder large SUV</a> in New York, ahead of the regular model&rsquo;s local debut later this year.</p>  <p>  	The Pathfinder Hybrid swaps the regular model&rsquo;s 3.5 litre petrol V6 for a supercharged 2.5 litre four cylinder, paired with a single electric motor and lithium-ion battery system. </p>  <p>  	This electric motor is small by hybrid standards (producing just 15kW), but combines with the petrol engine to produce 187kW/329Nm.</p>  <div class="inarticle-ad" style="text-align: center;">		<div id="ad-521d387bd4721" style="height: 250px;">
			<noscript><a href="http://ad.au.doubleclick.net/jump/tmr.news/details;car=nissanpathfinder;sz=300x250;tile=3;cat=suv;ord=33155346?" target="_blank"><img src="http://ad.au.doubleclick.net/ad/tmr.news/details;car=nissanpathfinder;sz=300x250;tile=3;cat=suv;ord=33155346?" border="0" alt=""/></a></noscript>
		</div>
</div>  <p>  	These figures nearly match the 195kW/336Nm of the regular 3.5 litre unit, but consumes a claimed (US-data) 24 percent less fuel on the combined cycle.</p>  <p>  	Available in front or all-wheel drive, the Pathfinder Hybrid balanced petrol and electric power via a dual clutch system, before sending power to the wheels through a CVT auto.</p>  <p>  	The lithium-ion battery system is cleverly packaged beneath the second-row seating, maintaining easy access to the seven-seater&rsquo;s third row.</p>  <p>  	
	<div class="inline-image-center">
	<a href="/content/image/2/0/2014_nissan_pathfinder_hybrid_06-0328.jpg" class="lightbox"><img alt="" src="/content/image/2/0/2014_nissan_pathfinder_hybrid_06-0328-460x272.jpg" class="small"/></a>
		</div>
</p>  <p>  	Visually, the Pathfinder Hybrid is distinguished only by specific badges and LED taillights, while the interior gains a energy consumption display within the existing multimedia screen.</p>  <p>  	The Pathfinder Hybrid will be available in US showrooms from mid-year, carrying a price premium of about ten percent over the regular model&rsquo;s US$28,650 (AU$27,432) starting price.</p>  <h3>  	Australia</h3>  <p>  	Speaking with TMR today, Nissan Australia&#39;s Peter Fadeyev said that it is too soon to say whether Australia will see the hybrid version of the new Pathfinder, due here in the third quarter of the year.</p>  <p>  	He added that drivetrain options are yet to be locked in for the local version, suggesting that a variant using the existing 3.0 litre turbodiesel V6 may also be on the cards.</p>  <ul class="list-style-more"><li><h4>Related News &amp; Reviews at TMR <span>&#9660;</span></h4></li>  	<li>  		<a href="http://www.themotorreport.com.au/tag/nissan-pathfinder">Pathfinder </a>| <a href="http://www.themotorreport.com.au/tag/nissan">Nissan </a>| <a href="http://www.themotorreport.com.au/tag/suv">SUVs </a>| <a href="http://www.themotorreport.com.au/tag/hybrid">Hybrids </a>| <a href="http://www.themotorreport.com.au/tag/family">Family Cars</a></li>  </ul>  <p>

	$("#comment-placeholder, #register_name, #register_email").bind("click focus", function() {
		var $placeholder = $("#comment-placeholder");
		var $real = $("#comment-real");

		if ($placeholder.is(":visible")) {
			$placeholder.hide();
			$real.slideDown("fast", function() {
				if (document.activeElement == window.document.getElementById("register_name") ||
					document.activeElement == window.document.getElementById("register_email")) {
					return;
				}
				$("#comment_text").focus();
			});
		}
	});

	$(".emoticons-container a").click(function(evnt) {
		evnt.preventDefault();

		var text = $(this).attr("data-emoticon");
		var textarea = document.getElementById("comment_text");

		var currSelectionStart = textarea.selectionStart;
		var currText = textarea.value;
		currText = currText.substring(0, textarea.selectionStart) +
		           text + currText.substring(textarea.selectionEnd);
		textarea.value = currText;

		$(textarea).focus();
		textarea.selectionStart = currSelectionStart+text.length;
	});

	$('#comment-form').ajaxForm({
		type: 'post',
		beforeSubmit: function(request) {
			$("#comment-error").empty().hide();

			$("#comment-form-wrapper .indicator").hide().html(loading_html).fadeIn();
			$("#comment-form input[type=image]").prop("disabled", true);
		},
		success: function(data) {
			$("#comment-reply-to").empty();
			$("#comment-form input[type=image]").prop("disabled", false);

			if (data.indexOf("ERROR:") === 0) {
				data = data.substr(6);
				$("#comment-error").html(data).show();
			} else {
				var parentId = $("#comment_parent_id").val();
				if (parentId != "" ) {
					var container = $("#comment_" + parentId).parent("li");

					// figure out what level the new comment is at
					var level = container.parents("ul").get(0).className;
					level = level.split("-")[2];
					level = (1*level) + 1;

					container.append("<ul class=\"comments-level-"+level+"\"><li>"+data+"</li></ul>");
				} else {
					$("#comments").append("<ul><li>" + data + "</li></ul>");
				}

				// if they've unchecked "Remember me", make sure to reset that part of the
				// form as well
				if (!$("#register_remember").prop("checked")) {
					$("#register_remember").removeAttr("checked");
					$("#register_name").attr("value", "");
					$("#register_email").attr("value", "");
				}

				FB.XFBML.parse();
				$("#comment-form").resetForm();

				var url = window.location.href.replace(window.location.hash, "");
				var regex = /id="comment_([0-9]+)"/i;
				var m = regex.exec(data);
				if (m.length > 1) {
					url += "#comment_"+m[1];
					window.location.href = url;
				}
			}
		}
	});

	var hash = window.location.hash.replace("#", "");

		$("#comment-form-header a." + hash).trigger("click");
